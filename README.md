# Inner Seasons

A menstrual cycle tracking app that visualizes your cycle through seasonal metaphors (Winter, Spring, Summer, Autumn), helping you understand and track your hormonal phases alongside daily wellness metrics.


## Overview
Inner Seasons addresses the issue of fragmented cycle tracking by consolidating multiple health metrics into a single, cohesive experience. Instead of using separate apps for mood, sleep, hydration, and period tracking, Inner Seasons connects everything within an intuitive seasonal framework.


## Key Features
### Daily Cycle Log
- **Period Tracking**: Simple toggle for menstruation days
- **Mood Monitoring**: Track your emotional state throughout the cycle
- **Sleep Tracking**: Log hours of sleep each night
- **Pain Level**: Record and monitor discomfort levels
- **Water Intake**: Track daily hydration

### Profile & Cycle Information
- **Last Period Start Date**: Input and track cycle beginnings
- **Average Cycle Length**: Personalized cycle duration tracking
- **Average Bleeding Duration**: Monitor typical period length
- **Current Phase**: Real-time display of your current seasonal phase
